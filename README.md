# zhweb-grandlisboa-resource

## 项目简介

本项目是一个面向 Web 技术归档的通用资源仓库，用于存放和发布多个独立 HTML 页面。页面内容涵盖前端演示、交互原型、静态信息页面等类型，不针对任何特定域名或网站。

仓库旨在为开发者、设计师以及技术爱好者提供一个集中存放与分享 HTML 页面资源的空间。所有页面均为独立文件，可直接在浏览器中打开运行，便于查看、测试与分发。

## 目录说明

```
zhweb-grandlisboa-resource/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面引用的公共资源（CSS、JS、图片等）
├── docs/           # 相关文档或说明文件（可选）
└── README.md       # 本文件
```

- **pages/**：包含多个独立的 `.html` 文件，每个文件为一个完整的页面。
- **assets/**：存放页面共用或专用的样式、脚本、图片等静态资源。
- **docs/**：可放置项目说明、使用指南或其他辅助文档。

## 页面归档说明

- 所有页面均以独立文件形式存放，不依赖外部动态服务或数据库。
- 每个页面文件名应反映其内容或用途，建议使用英文或拼音命名，便于识别。
- 页面之间相互独立，不存在互相依赖关系，可直接通过文件路径访问。
- 如需引用公共资源，请将资源文件放入 `assets/` 目录，并在页面中正确引用相对路径。

## 维护说明

- 欢迎提交新页面或改进现有页面，请将文件放入 `pages/` 目录，并确保链接和资源引用正确。
- 提交前请确认页面内容不包含任何具体域名、网址、营销信息或诱导点击内容。
- 请保持目录结构清晰，避免冗余文件。
- 如有较大改动或新增功能，建议在提交时附带简要说明。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 打开 `pages/` 目录下的任意 `.html` 文件，即可在浏览器中查看页面。
3. 如需部署到 Web 服务器，请将整个仓库或 `pages/` 目录上传至服务器根目录或相应路径。

## 注意事项

- 本仓库仅用于技术归档与分享，不涉及任何商业用途或特定网站推广。
- 页面内容应遵守相关法律法规，不得包含违法违规信息。
- 仓库维护者保留对内容进行审查和调整的权利。
